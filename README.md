## Task Manager

### Main Features:

- Check-in / Check-out to unlock the management of tasks
- Visualize tasks
- Mark tasks as finished

### Additional Features (Future)

- Signup / Sign-in
- Filter tasks by date (or range of dates)
- Send Pictures on tasks

### Front-End Technologies

- React-Native
- Typescript
- Styled-Components
- Redux-Saga
- Watermelon DB
- React-Navigation
